# Изучаем git

## 1. Знакомство с Git Commit

- [x] Коммиты в GIT:

```sh
git commit
git commit
```

- [x] Ветвление в Git:

```sh
git branch
git checkout 
```

- [x] Ветки и слияния:

```sh
git branch 
git checkout 
git commit
git merge
```

- [x] Git Rebase

```sh
git branch
git checkout
git commit
git rebase 
```
